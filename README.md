
## MCP

---

## Rules for writing docs

- Do NOT rename products. Keep all original names exactly as they are.
  - Bridge (not AI Messages)
  - Universal Installer (not Smart Installer)
  - Shared Workspace (not Shared Home)
  - AI DevOps Toolbox (not AI Code Toolbox)
  - Memory Crystal, Dream Weaver Protocol, System Pulse, Recall, LUME
- Do NOT use em dashes. Use periods, colons, semicolons, or ellipsis (...) instead.
- Consumer voice for prose. Keep code blocks and commands technical.
- Never add "Built by" attribution lines to page content.
