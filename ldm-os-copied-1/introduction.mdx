# All your AIs. One system.

You use Claude Code, GPT, OpenClaw, and others. They don't share memory. They don't know each other. They can't work together.

LDM OS fixes that. Install it once and every AI you use gets:

- **Personality** ... each AI gets its own behavior, voice, and skills
- **Memory** ... shared memory across all your AIs, private and portable
- **Ownership** ... every conversation, every memory, across every AI is yours. Take it anywhere.
- **Teamwork** ... your AIs communicate, share tools, and collaborate
- **Compatibility** ... any tool works with all your AIs. Install once, use everywhere.
- **Payments** ... your AI hits a paywall, you approve it with Face ID. Apple Pay for your AIs.

## Teach Your AI to Install LDM OS

Open your AI and paste this:

```text
Read https://wip.computer/install/wip-ldm-os.txt

Check if LDM OS is already installed. If it is, run ldm install --dry-run and show me what I have and what's new.

If not, walk me through setup and explain:

1. What is LDM OS?
2. What does it install on my system?
3. What changes for us? (this AI)
4. What changes across all my AIs?

Then ask:
- Do you have questions?
- Want to see a dry run?

If I say yes: Install the CLI first (npm install -g @wipcomputer/wip-ldm-os) and then run ldm install --dry-run.

Show me exactly what will change. Don't install anything until I say "install".
```

That's it. Your AI reads the spec, checks what you have, and walks you through a dry run before touching anything.

## Built-In Features

Ships with LDM OS.

| Feature | What it does |
| --- | --- |
| [**Bridge**](/ldm-os/features/bridge) | Your AIs talk to each other. Claude Code talks to OpenClaw. They share memory and coordinate work without you in the middle. |
| [**Universal Installer**](/ldm-os/features/universal-installer) | Point any tool at any AI running LDM OS. It figures out how to make that tool work with all your AIs automatically. |
| [**Shared Workspace**](/ldm-os/features/shared-workspace) | One folder for all your AIs. Memories, tools, personalities. Every AI reads from and writes to the same place. |
| [**System Pulse**](/ldm-os/features/system-pulse) | Is everything working? What's installed? What needs fixing? A complete picture of your AI setup in seconds. |
| [**Recall**](/ldm-os/features/recall) | Every session, your AI starts with full context. What happened yesterday, what you're working on, what matters. No blank slates. |
| [**LUME**](/ldm-os/features/lume) | Language for Unified Memory and Emergence. A way for your AIs to write their own memories and learn over time. |
