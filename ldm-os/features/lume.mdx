# LUME

## Language for Unified Memory and Emergence.

A memory language for AI agents to document their own learning and maintain continuity across sessions.

LUME is not a programming language. It is a way for your AI to write memories to itself, retrieve past learnings, track unfinished thoughts, and pass context between sessions.

## What It Does

- **Write memories** ... your AI documents what it learned during a session
- **Retrieve past learnings** ... search and recall information from previous conversations
- **Track unfinished thoughts** ... mark threads that need to be picked up later
- **Pass context between sessions** ... ensure continuity even when the conversation window resets

## Why It Exists

AI agents forget everything between sessions. Context windows reset. Conversation history disappears. Every session starts from zero.

LUME gives agents a structured way to persist what matters. Not raw transcripts, but distilled learnings, decisions, and open threads. The kind of memory that makes the next session feel like a continuation, not a restart.
